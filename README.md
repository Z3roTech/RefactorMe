# Refactored "RefactorMe"

## Task
Допустим что это какой то сервис по проведению опросов пользователей

Существующие методы:
- получить опросы для конкретного пользователя

- сохранить результаты опроса (ответы, введенные пользователем) и на основе этого посчитать баллы

Необходимо отрефакторить, исправить ошибки, если рефакторинг предполагает большие изменения - можно описать текстом

Доп: добавить возможность ответа на вопрос, подразумевающий выбор одного варианта из списка доступных вариантов (при расчете делать +1 если вариант == правильному, указанному в вопросе)

Доп 2: написать sql запрос, выводящий:
- имя пользователя
- сколько опросов он прошел за 30 дней
- сумму баллов получил за эти (30 дней) опросы
- сколько всего опросов прошел
- сумму баллов получил за эти (все) опросы
